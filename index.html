GXD High Throughput GEO Load

See GEO Load Wiki for input file format, report descriptions, processing algorithms, etc.

Input

  • GEO Input Directory
    • geo.xml.* - files of 5000 experiments
    • GSExxxxx_family.xml - files of samples for a given experiment
  • geo.uid.xml -- This is the file of GEO UIDs and meta data used to fetch the experiments

Output Files

  • BCP File directory
  • List of experiments in the DB that are not evaluated; used by the classifier
  • The classifier predictions; used to update the experiment predicted evaluation state and confidence

Logs

  • Sample Mirror Curator Log
  • Experiment Mirror Log
  • Sample Mirror Log
  • Process Log
  • Diagnostic Log
  • Classifier Prediction Log

Reports

  • GEO QC Report - load stats
  • Parsing reports
    • Experiment Parsing Report
    • Sample Parsing Report
    • Samples In Database Parsing Report
    • Curated Experiment with changed Samples Report

GXD High Throughput ArrayExpress Load

See ArrayExpress Load Wiki for input file format, report descriptions, processing algorithms, etc.

Input

  • ArrayExpress Input Directory
    • experiment.txt - Connie's list of experiment IDs she would like loaded
    • .json - the BioStudies experiment file for "exptID"
    • .sdrf.txt - the BioStudies sample file for "exptID"

Output Files

  • BCP File directory

Logs

  • Mirror Curator Log - files that could not be downloaded
  • Mirror Log - the wget commands for each file with the status of the download
  • Process Log
  • Diagnostic Log
  • Curator Log - stats and list of any experiments that could not be loaded

Reports

  • ArrayExpress Curator File QC Report
  • Experiment Parsing Report
